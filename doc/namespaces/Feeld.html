Interfaces

FieldInterface Classes that implement FieldInterface (short: Fields) define types of data entry (e. g. by selecting one of several values, by checking a box or by inputting a text). (Please note that this has nothing to do with the UI of the Field, "selecting one of several values" might be done by clicking on the value or typing it in; for the UI-component of Feeld, see DisplayInterface)
